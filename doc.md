### Функции сервиса
1. Добавление групп.
2. Получение списка активных групп по параметрам:
   a.По году поступления(group_year < n-ого года, group_year = n-ый год, group_year > n-ого года),
   b.По id руководителя,
   c.По количеству студентов(group_stud_count < n-ого кол-во студентов, group_stud_count = n-ому кол-ву студентов, group_stud_count > n-ого кол-во студентов),
   d.По шифру направления подготовки,
   e.По номеру группы,
   f.По тому после какого класса поступили.
4. Получение информации о определенной группе группе.
5. Изменение статуса группы на нективную(Удаление).
6. Изменение информации о определенной группе
   a.Изменение руководителя,
   b.Изменение количества студентов.

### Функции сервиса
1. Добавление группы.
    Добавление группы включает в себя:
    Запись года создания группы (year < 3000 and year > 2000)
    Запись id руководителя (ruk_id > 0)
    Запись количества студентов (stud_count > 0, stud_count < 30)
    Запись шифра направления подготовки (Вида строго NN.NN.NN)
    Запись номера группы (number > 0)
    Запись после какого класса поступили(Строго 9 или 11)
    Запись префикса (len_prefix > 0)
2. Получение списка активных групп по параметрам:
    По году поступления(group_year < n-ого года, group_year = n-ый год, group_year > n-ого года),
    По id руководителя,
    По количеству студентов(group_stud_count < n-ого кол-во студентов, group_stud_count = n-ому кол-ву студентов, group_stud_count > n-ого кол-во студентов),
    По шифру направления подготовки,
    По номеру группы,
    По тому после какого класса поступили.
3. Получение информации о группе.
    С помощью этой функции можно получить следующую информацию о группе: Год поступления, id_руководителя, количество студентов, шифр направления подготовки, номер группы, после какого класса поступили студенты этой группы и префикс
4. Удаление группы
    Подрозумевает под собой изменение статуса активности группы на неактивную
5. Изменение информации о определенной группе
    Изменение руководителя,
    Изменение количества студентов.
